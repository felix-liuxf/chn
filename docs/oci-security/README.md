##### ISO 27018 证书
ISO/IEC 27018:2014基于ISO/IEC 29100的隐私原则，建立了一套适用于公有云服务的个人隐私保护规范，包含控制目标、控制内容以及相应的实施指引。

ISO/IEC 27018:2014特别考量相关法律与规范，依据ISOIEC 27002识别出的公有云服务信息安全风险环境，规定适用的个人可识别信息保护要求。
